# ContextAtlas 文档总览

本文档是 `docs/` 的统一入口。

文档按生命周期和用途分成 6 类：

- `guides/`
  面向使用者的稳定教程与上手路径。
- `reference/`
  CLI、MCP 等稳定参考文档。
- `architecture/`
  仓库定位、工程定位、架构图等长期说明。
- `product/`
  产品路线图与版本演进方向。
- `project/`
  项目级知识说明，如项目记忆模型。
- `archive/`
  带日期的交付、验收、迭代过程文档归档。
